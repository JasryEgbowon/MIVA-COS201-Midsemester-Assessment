## Quick Start to use this code

### Prerequisites- You need these before you can even edit this code 
- GCC Compiler (MinGW for Windows)
- Or any C IDE (Code::Blocks, Dev-C++, CLion)
### Installation & Run
```bash
#download the Zipfolder from GitHub, unzip it and then:

# Compile the program
gcc -o student_system student_record_system.c

# Run the application
./student_system.exe
```
